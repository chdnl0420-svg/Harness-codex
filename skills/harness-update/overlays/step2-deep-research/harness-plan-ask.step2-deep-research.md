## CRITICAL: Step 2 Ask Mode Research Branch

`harness-plan-ask` inherits the same Phase 2 research branch as `harness-plan`.

After collecting the six interactive categories, the caller must check whether any answer requires research. If yes, use `harness-deep-researcher` or direct research before drafting the domain plan. If no, record the reason in progress.

Ask mode changes input collection only. It does not remove Step 2 deep research.
