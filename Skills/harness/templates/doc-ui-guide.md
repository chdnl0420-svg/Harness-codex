# UI Guide

> 이 문서는 **"어떻게 보여야 하는지"** 정의한다 (선택 사항).
> 웹/모바일/CLI UI 가 있는 프로젝트에서만 채워넣음. 백엔드 only 면 비워둬도 OK.
>
> 작성하려면: `/harness-spec ui` (대화형).
