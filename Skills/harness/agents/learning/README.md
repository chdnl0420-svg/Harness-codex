# Harness Learning Files

이 폴더는 Codex용 Harness 도우미가 반복 작업에서 배운 운영 규칙을 보관한다.

## 위치

```text
~/.codex/skills/harness/agents/learning/<agent>.md
```

프로젝트별 `.harness`에는 필요한 경우 복사본만 둔다. 정본은 Codex 스킬 폴더다.

## 작성 규칙

- 날짜가 있는 짧은 항목으로 쓴다.
- 실제 검증에서 반복 확인된 규칙만 남긴다.
- 특정 프로젝트 비밀, 토큰, 계정 정보는 쓰지 않는다.
- 실행 도구 이름은 Codex에서 실제 사용할 수 있는 이름으로 쓴다.

## 보존 기준

- 실패를 줄이는 구체 절차는 보존한다.
- 특정 도구의 이름보다 선택 기준이 중요하면 선택 기준으로 바꿔 쓴다.
- 논문/블로그 인용은 규칙의 근거가 될 때만 남긴다.
- 너무 긴 항목은 “상황 → 판단 → 행동” 3문장으로 압축한다.
- 삭제가 필요한 항목은 바로 지우지 말고 먼저 `Deprecated`로 표시한다.
